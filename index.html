¿Cómo se hacen las páginas web?

Comencemos . . .

Todos hemos entrado alguna vez a una página web o puede que incluso las usemos en nuestra vida cotidiana, ya sea por el trabajo, la escuela o el ocio. Son bastante útiles debido a la información que algunas nos pueden proporcionar podemos informarnos un determinado tema, usar las herramientas que nos brindan o entretenernos.

Sin embargo, sabemos muy poco con respecto a su funcionamiento o del proceso que hay para crearlas, subirlas a internet y los conocimientos sobre tecnologías y lenguajes para hacerlo.

En este texto se tratara de abarcar el como y lo que conlleva crear una página de internet profundizando sólo lo suficiente como para tener una idea clara del proceso y del porqué de ciertas cosas, a la vez que se hace de manera comprensible para cualquiera y así poder conocer más sobre el funcionamiento de lo que usamos. Podría ser servirte si te interesa el desarrollo web o bien puede simplemente ser algo interesante al tener la curiosidad de conocer y descubrir algo nuevo.

¿Qué es realmente una página web?

Una página web básicamente es un documento, específicamente un documento HTML, en dicho documento se escribe código HTML (HyperText Markup Language), el cual se usa para representar elementos que estructuran la información de la página, por ejemplo: textos, hipervínculos (links), imágenes, vídeos, tablas, formularios, contenedores de información, etc.

El documento puede ser subido a internet y así formar parte de la World Wide Web o simplemente web, que es el nombre con el que se conoce al conjunto de páginas subidas a internet que existen. En resumen una página web es un documento HTML que está en la web.

Este tipo de documento puede ser leído por un navegador web, existen varios de éstos: Google Chrome, Mozilla Firefox, Microsoft Edge, Opera, Safari y el recientemente popularizado Brave, entre otros.

Todos estos son programas que al leer el código HTML pueden mostrarnos la información con el formato indicado en dicho código, ósea mostrarnos la página web. En el código está toda la información, la cual debe estar estructurada de manera que el navegador pueda saber que es cada cosa: títulos, párrafos, enlistados, cabeceras, contenido principal, contenido secundario, pie de página, etc.

Junto a esto, los navegadores también permiten localizar una página específica entre la web, esto por medio de una dirección llamada URL, que es una dirección única para cada página.

Lenguaje HTML

Como ya se mencionó, el documento HTML debe contener código en lenguaje HTML, en éste código, principalmente, se marca toda la estructura de la página. Las reglas y patrones para escribir este código (sintaxis) se basan en etiquetas las cuales son la representación de los elementos de la página. Ver articulo completo
